# Talent-Base-Front

## Pré-Requisitos

- Node
- npm

## Execução

Clonar repositório

`git clone https://github.com/Talent-Base/Talent-Base-Front.git`

Instalar as dependências

`npm install`

Rode o comando para executar a aplicação:

`npm run dev`

Por fim, acesse a porta do frontend em:

`http://localhost:3000/`

## Executando em Docker

`docker compose up --build`
